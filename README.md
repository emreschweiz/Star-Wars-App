# Star Wars App

SWAPI kullanarak yıldız gemilerini listeleyen, arama yapan ve detay sayfasına yönlendiren bir Vite + React uygulaması.

## Özellikler
- Yıldız gemilerini listeleme
- Ad veya modele göre arama
- “Daha Fazla” ile sayfalama
- Detay sayfası (ad, model, yolcu, hız, üretici, mürettebat, kargo)
- Kartlarda uzay gemisi görselleri

## Kullanılan Kaynaklar
- Veri kaynağı: SWAPI (`https://swapi.dev`)
- Görseller: Wookieepedia (Fandom) `Category:Starships` (MediaWiki PageImages API)
- Ek görseller: StarWars.com Databank (`https://www.starwars.com/databank`)
- Görsel yedeği: Pixabay (`https://pixabay.com`)

## Kurulum
```bash
npm install
```

## Geliştirme

